# Simple Python Calculator

## Overview

This is a basic calculator project written in **Python**. The program allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division.

This project is useful for beginners who want to understand basic programming concepts like user input, conditional statements, and arithmetic operations.

## Features

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division of two numbers
* Prevents division by zero errors
* Simple command-line interface

## Technologies Used

* Python 3

## How It Works

1. The program displays a menu with four operations.
2. The user selects an operation by entering a number (1–4).
3. The user enters two numbers.
4. The program performs the selected operation and displays the result.

## Example Output

```
Simple Calculator
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter choice (1/2/3/4): 1
Enter first number: 5
Enter second number: 3
Result = 8
```

1. Install Python on your computer.
2. Save the calculator code in a file named `calculator.py`.
